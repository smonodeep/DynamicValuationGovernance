how that flows through governance frameworks to finally create the decisions.
Though the initial focus is on Property Lending with focus on multifamily apartments.
which is an intersection of urban economics, 
Commercial Real Estate(CRE) and Residential Real Estate(RRE) 
Data Sources and broad method
The Raw sources for the data are
HPI- Raw HPI extracted from RBI. Since the Series was across multiple bases.
Cross linking of the series is done by base normalization
Rent-Raw Rent file for individual city is manually extracted from CREMatrix across cities.
Repo- Repo Rate is extracted from FRED. Final data is from 2015Q1 to 2024Q4 across Bangalore, Chennai, Delhi, Mumbai
